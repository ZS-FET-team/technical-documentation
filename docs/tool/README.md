## ESLint + Prettier

ESLint 是一款插件化的 JavaScript 代码静态检查工具 ，只能对 js 代码进行检查和代码格式化

Prettier 是一个代码格式化工具，可以格式化（html、css、js、ts、vue、markedown）等前端代码，但不具备代码检查功能

::: tip 推荐
ESLint 和 Prettier 一起结合起来使用，用 ESLint 进行代码校验，用 Prettier 统一代码风格。
:::
